# Piano Samples

These local piano samples are used by the Enchorder `Tone.Sampler` piano instrument.

## Source

- Repository: `nbrosowsky/tonejs-instruments`
- Piano sample directory: `samples/piano`
- Sample source info from that repository lists piano as `VSO2`.

## License

- License URL: https://creativecommons.org/licenses/by/3.0/
- Code license for the source repository is MIT, but these audio files are attributed as samples under CC-BY 3.0.

## Included Files

- `C4.mp3`
- `Ds4.mp3`
- `Fs4.mp3`
- `A4.mp3`
- `C5.mp3`

These are intentionally sparse. Tone.js pitch-shifts from these mapped samples to play nearby notes.
