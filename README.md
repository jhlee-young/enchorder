# enchorder

Minimal chord learning workbench for hearing chord progressions and seeing how each chord is built.

## Features

- Type chord progressions like `Cmaj7 | Am7 D7 | Gmaj7`.
- See each chord's root, type, notes, and short beginner-friendly explanation.
- Play the full progression or preview one chord at a time.
- Learn common chord types from the compact Chord Library.
- Adjust `BPM` while learning the term as part of the interface.

## Development

```sh
npm install
npm run dev
npm test
npm run build
```

The UI copy is written in English, and visual styling is centralized in `src/styles/theme.css`.
